### truncate

```python
truncate(unit)
```

Truncate the expression to a time expression in units of `unit`.

Commonly used for time series resampling.

#### Parameters

| Name | Type                                                               | Description             | Default    |
|------|--------------------------------------------|---------------|--------|
| unit | [Literal](%60typing.Literal%60)\['h', 'm', 's', 'ms', 'us', 'ns'\] | The unit to truncate to | *required* |

#### Returns

| Name | Type                                                                | Description                |
|-------|---------------------------------------------|--------------------|
|      | [TimeValue](%60letsql.vendor.ibis.expr.types.temporal.TimeValue%60) | `self` truncated to `unit` |

